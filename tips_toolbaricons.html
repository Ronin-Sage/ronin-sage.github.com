:: Custom Firefox Bookmark-toolbar Icons ::

Chances are, if you're an avid Firefox user, you have some folders of your own placed on its bookmark toolbar. By default, these folders use the typical vanilla folder icon, but there's actually a way to edit these icons. There are several ways to accomplish this, including an add-on to manage the icons easily, but here we'll focus on the "under-the-hood" method. But first, some file locations to note:

Linux:
~/.mozilla/firefox/[profile]/chrome/userChrome.css

Windows:
../Documents and Settings/[user]/Application Data/Mozilla/Firefox/Profiles/[profile]/chrome/userChrome.css

Using a text editor, open up your userChrome.css file(create it if it does not exist).

It's a standard CSS(Cascading Style Sheets), so create a section for the -moz-image-region class, defining the attributes like so:

.bookmark-item[container="true"][label="The Barracks"]
{
 list-style-image: url('/home/ronin-sage/ico/soldier.png') !important;
 -moz-image-region: rect(0px 16px 16px 0px) !important;
}

The label value should be whatever the name of the folder is(in this case, "The Armory" for example).

list-style-image's url value is the location of the image you wish to use (I recommend using PNG for transparency purposes).

The -moz-image-region is the geometric region of the image file used to create the mini-icon, drawn from the upper-left in this case. While you can use any size image, provided that you use the appropiate region(for instance, "128px" four times/once for a 128x128 image), I'm somewhat sure that the quality takes a hit with the larger files, so I stuck to 16x16 as my standard here.
