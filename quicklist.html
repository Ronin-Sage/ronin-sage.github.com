++The Quick-List++

A list of useful "underground" free/FOSS
apps I've come across in my time.

Name: Midori
Platforms: Linux, others(dependency-based)
GUI: GTK+
License: LGPL(free/open-source)
Description: Very promising lightweight WebKit browser
Homepage: http://software.twotoasts.de/
Rating: 4/5 win-points
Name: Iron
Platforms: Windows
License: BSD(free/open-source)
Description: Webkit and Chromium-based browser, with the privacy concerns of the latter resolved
Homepage: http://www.srware.net/en/software_srware_iron.php
Rating: 4.5/5 win-points
Name: Paint.NET
Platforms: Windows
License: Freeware, MIT
Description: Lightweight raster-graphics editor with plugin support
Homepage: http://www.getpaint.net/
Rating: 4.5/5 win-points
Name: gmrun
Platforms: Linux
GUI: GTK+
License: GPL(free/open-source)
Description: Light "run"/"completion-run" utility, with history searches and URL handlers
Homepage: http://sourceforge.net/projects/gmrun
Rating: 4.5/5 win-points
Name: yeahconsole
Platforms: Linux
License: GPL(free/open-source)
Description: Terminal emulator wrapper that allows for a quake/fps-console "drop-down" feel
Homepage: http://phrat.de/yeahtools.html
Rating: 4/5 win-points
Name: Komodo Edit
Platforms: Linux, Windows, OS X
GUI: GTK+(Linux)
License: MPL, GPL, LGPL(free/open-source)
Description: Middle-weight web/scripting language IDE/editor
Homepage: http://www.activestate.com/komodo_edit
Rating: 4/5 win-points
Name: Geany
Platforms: Linux, Windows, OS X, & others
GUI: GTK+
License: GPL(free/open-source)
Description: Light, but powerful code-editor with plugin support
Homepage: http://www.geany.org/
Rating: 4.5/5 win-points
Name: Go-OpenOffice
Platforms: Windows, Linux, OS X
GUI: GTK+(Linux), [OS native]
License: GPL,CDDL(free/open-source)
Description: Office suite forked from OpenOffice featuring improved performance, better interoperability, specific framework support(Mono, GStreamer), and other enhancements
Homepage: http://go-oo.org
Rating: 5/5 win-points
Name: Abiword
Platforms: Windows, Linux, OS X(a bit outdated), & others
GUI: GTK+, [OS native]
License: GPL(free/open-source)
Description: Lightweight, yet impressively-featured rich text editor
Homepage: http://www.abisource.com/
Rating: 5/5 win-points
Name: GNOME Color Chooser
Platforms: Linux, others(anything that can run GTK+)
GUI: GTK+
License: GPL(free/open-source)
Description: Advanced color/engine configuration utility for GTK+ themes
Homepage: http://gnomecc.sourceforge.net/
Rating: 4.5/5 win-points
Name: HardInfo
Platforms: Linux
GUI: GTK+
License: GPL2(free/open-source)
Description: System information and CPU-benchmarking tool
Homepage: http://hardinfo.berlios.de/HomePage
Rating: 4.5/5 win-points
Name: Baku
Platforms: Windows
License: MPL(free/open-source)
Description: Junkfile-remover and registry cleaner/compactor
Homepage: http://pmcchp.com/baku/index.html
Rating: 4/5 win-points
Name: MyDefrag
Platforms: Windows
License: Freeware
Description: Thorough disk defrag/optimization tool
Homepage: http://www.mydefrag.com
Rating: 4.5/5 win-points
Name: Quintessential Player(QMP; formerly, QCD)
Platforms: Windows
License: Freeware
Description: Quite powerful and customizable media player(handles video, but primarily meant for audio)
Homepage: http://www.quinnware.com/
Rating: 4.5/5 win-points
Name: AIMP
Platforms: Windows
License: Freeware
Description: Very memory-efficient, feature-rich alternative audio player
Homepage: http://aimp.ru/index.php?do=lang-en
Rating: 5/5 win-points
